# <a name="labor-cost-tracking-on-projects"></a><span data-ttu-id="b614d-103">Suivi des coûts de main-d’œuvre sur les projets</span><span class="sxs-lookup"><span data-stu-id="b614d-103">Labor cost tracking on projects</span></span>

<span data-ttu-id="b614d-104">_**S’applique à :** Project Operations pour les scénarios basés sur les ressources/produits non stockés Déploiement simplifié – Traiter la facturation pro forma_</span><span class="sxs-lookup"><span data-stu-id="b614d-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios, Lite deployment - deal to proforma invoicing_</span></span>

<span data-ttu-id="b614d-105">Dynamics 365 Project Operations suit les estimations de main-d’œuvre et les dépenses avec la plus faible précision requise sur le plan de projet.</span><span class="sxs-lookup"><span data-stu-id="b614d-105">Dynamics 365 Project Operations tracks labor estimates and spend at the lowest required granularity on the project plan.</span></span> <span data-ttu-id="b614d-106">L’estimation financière du coût de la main-d’œuvre est basée sur l’effort planifié et les ressources génériques ou nommées affectées à chaque tâche de nœud terminal dans le plan de projet.</span><span class="sxs-lookup"><span data-stu-id="b614d-106">The financial estimate of labor cost is based on the planned effort and the generic or named resources that are assigned to each leaf-node task in the project plan.</span></span> <span data-ttu-id="b614d-107">Lorsque le projet commence et que les gens commencent à indiquer le temps passé sur diverses tâches du projet, les dépenses réelles pour la main-d’œuvre sont résumées, ce qui lance un calcul des projections.</span><span class="sxs-lookup"><span data-stu-id="b614d-107">When the project begins and people start to report time on various project tasks, the actual spend on labor is summarized which starts a calculation of the projections.</span></span>

## <a name="labor-cost-tracking-view"></a><span data-ttu-id="b614d-108">Vue du suivi des coûts de main-d’œuvre</span><span class="sxs-lookup"><span data-stu-id="b614d-108">Labor Cost Tracking view</span></span>

<span data-ttu-id="b614d-109">Sur la page **Projets**, sur l’onglet **Suivi**, vous pouvez sélectionner **Suivi** > **Coût** pour ouvrir la vue **Suivi du coût** et voir la progression des dépenses de main-d’œuvre pour chaque tâche dans le plan de projet.</span><span class="sxs-lookup"><span data-stu-id="b614d-109">On the **Projects** page, on the **Tracking** tab, you can select **Tracking** > **Cost** to open the **Cost Tracking** view and see the progress of labor spend on each task in the project plan.</span></span> <span data-ttu-id="b614d-110">Cette vue compare également le coût de main-d’œuvre réel consacré à une tâche au coût de main-d’œuvre planifié de la tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-110">This view also compares the actual labor cost spent on a task to the task's planned labor cost.</span></span> <span data-ttu-id="b614d-111">Project Operations utilise les formules suivantes pour calculer les mesures du coût de la main-d’œuvre :</span><span class="sxs-lookup"><span data-stu-id="b614d-111">Project Operations uses the following formulas to calculate labor cost metrics:</span></span>

- <span data-ttu-id="b614d-112">**Coût planifié** : Coût estimé de toutes les affectations de ressources sur chaque tâche de nœud terminal</span><span class="sxs-lookup"><span data-stu-id="b614d-112">**Planned cost**: Estimated cost of all resource assignments on each leaf node task</span></span>
- <span data-ttu-id="b614d-113">**Coût réel** : Somme de tous les chiffres réels de coût pour le temps enregistré sur la tâche</span><span class="sxs-lookup"><span data-stu-id="b614d-113">**Actual Cost**: Sum of all cost actuals for time recorded on the task</span></span>
- <span data-ttu-id="b614d-114">**Pourcentage de consommation du coût** : Coût réel ÷ Coût estimé à achèvement</span><span class="sxs-lookup"><span data-stu-id="b614d-114">**Cost consumption percentage**: Actual cost ÷ Cost estimate at complete</span></span>
- <span data-ttu-id="b614d-115">**Coût restant** : Coût estimé à achèvement – Coût réel</span><span class="sxs-lookup"><span data-stu-id="b614d-115">**Remaining Cost**: Cost estimate at complete  – Actual cost</span></span>
- <span data-ttu-id="b614d-116">**Coût à achèvement** : Coût restant + Coût réel</span><span class="sxs-lookup"><span data-stu-id="b614d-116">**Cost at Complete**: Remaining cost + Actual cost</span></span>
- <span data-ttu-id="b614d-117">**Écart de coût** : Coût planifié – Coût estimé à achèvement</span><span class="sxs-lookup"><span data-stu-id="b614d-117">**Cost variance**: Planned cost – Cost estimate at complete</span></span>

<span data-ttu-id="b614d-118">Chaque tâche montre une projection de l’écart de coûts sur la tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-118">Each task shows a projection of the cost variance on the task.</span></span> <span data-ttu-id="b614d-119">Si l’estimation des coûts à l’achèvement est supérieure au coût planifié, la tâche devrait dépasser le budget.</span><span class="sxs-lookup"><span data-stu-id="b614d-119">If the cost estimate at complete is more than the planned cost, the task is projected to go over budget.</span></span> <span data-ttu-id="b614d-120">Si l’estimation des coûts à l’achèvement est inférieure au coût planifié, la tâche devrait être en-dessous du budget.</span><span class="sxs-lookup"><span data-stu-id="b614d-120">If the cost estimate at complete is less than the planned cost, the task is projected to finish under budget.</span></span>

>[!NOTE]
> <span data-ttu-id="b614d-121">Project Operations n’affiche que les coûts de main-d’œuvre sur la page **Projet** de l’onglet **Suivi**. <span data-ttu-id="b614d-122">Tous les montants de coût indiqués sont convertis dans la devise des coûts du projet à partir de la devise de prix de revient du projet utilisée pour déterminer le taux de coût.</span><span class="sxs-lookup"><span data-stu-id="b614d-122">All cost amounts shown are converted to the project's cost currency from the project's cost price currency used to determine the cost rate.</span></span> <span data-ttu-id="b614d-123">La devise de coût du projet est la devise de l’unité contractuelle du projet.</span><span class="sxs-lookup"><span data-stu-id="b614d-123">The cost currency of the project is the currency of the contracting unit on the project.</span></span> <span data-ttu-id="b614d-124">Les valeurs de coût estimées pour le temps qui sont indiquées sur l’onglet **Estimations** de la page **Projet** peuvent ne pas correspondre au coût planifié sur l’onglet **Suivi**. The reason for this discrepancy is because of the differences in how estimated cost is summarized on the **Estimates** grid and how planned cost is calculated on the **Tracking** grid.</span></span> 
>
> - <span data-ttu-id="b614d-125">L’**onglet Estimations** calcule le coût estimé en utilisant la même devise que le taux de coût du tarif.</span><span class="sxs-lookup"><span data-stu-id="b614d-125">The **Estimates tab** calculates the estimated cost by using the same currency of the cost rate in the price list.</span></span> <span data-ttu-id="b614d-126">Ensuite, le coût estimé dans la devise du tarif est converti en coût estimé dans la devise de coût du projet.</span><span class="sxs-lookup"><span data-stu-id="b614d-126">Then, the estimated cost in the price list currency is converted to the estimated cost in the project's cost currency.</span></span> <span data-ttu-id="b614d-127">Le coût estimé dans la devise du projet est arrondi à 2 décimales.</span><span class="sxs-lookup"><span data-stu-id="b614d-127">The estimated cost in the project currency is shown rounded to 2 decimal places.</span></span> <span data-ttu-id="b614d-128">La précision de la devise n’est appliquée à aucun moment de cette conversion.</span><span class="sxs-lookup"><span data-stu-id="b614d-128">At no point during this conversion is currency precision applied.</span></span> 
> - <span data-ttu-id="b614d-129">Sur l’onglet **Suivi**, le calcul des coûts planifiés suit un ordre de calcul légèrement différent qui implique que la précision de la devise soit appliquée en deux étapes :</span><span class="sxs-lookup"><span data-stu-id="b614d-129">On the **Tracking** tab, the planned cost calculation follows a slightly different calculation order that involves currency precision being applied in two stages:</span></span> 
   ><ol>
   ><li><span data-ttu-id="b614d-130">Le montant du coût estimé dans la devise du tarif est converti dans la devise de base (conversion 1).</span><span class="sxs-lookup"><span data-stu-id="b614d-130">The estimated cost amount in the price list currency is converted to the base currency (conversion 1).</span></span></li>
   ><li><span data-ttu-id="b614d-131">Le montant du coût estimé dans la devise de base est converti dans la devise de coûts du projet (conversion 2).</span><span class="sxs-lookup"><span data-stu-id="b614d-131">The estimated cost amount in the base currency is converted to the project cost currency (conversion 2).</span></span> </li>
   ></ol>
   ><span data-ttu-id="b614d-132">La précision de la devise est appliquée au cours des deux étapes pour obtenir un coût planifié (sur l’onglet **Suivi**) qui s’écarte légèrement du coût estimé (sur la vue par **phases de temps** de l’onglet **Estimations**).</span><span class="sxs-lookup"><span data-stu-id="b614d-132">Currency precision is applied in both steps to get a planned cost (on the **Tracking** tab) that deviates slightly from the estimated cost (on the **Time - phased** view on the **Estimates** tab).</span></span> 
   
## <a name="reprojecting-costs-on-leaf-node-tasks"></a><span data-ttu-id="b614d-133">Reprojection des coûts sur les tâches de nœud terminal</span><span class="sxs-lookup"><span data-stu-id="b614d-133">Reprojecting costs on leaf node tasks</span></span>

<span data-ttu-id="b614d-134">Les coûts de main-d’œuvre sur une tâche de nœud terminal ne peuvent pas être directement reprojetés sur l’onglet **Suivi** de la **page Projet**.</span><span class="sxs-lookup"><span data-stu-id="b614d-134">Labor costs on a leaf node task can't be directly reprojected on the **Tracking** tab on the **Project page**.</span></span> <span data-ttu-id="b614d-135">Cependant, vous pouvez utiliser la vue **Suivi de l’effort** pour reprojeter l’effort restant sur une tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-135">However, you can use the **Effort Tracking** view to reproject the remaining effort on a task.</span></span> <span data-ttu-id="b614d-136">Cela provoque un recalcul du coût restant de la tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-136">This causes a recalculation of the remaining cost on the task.</span></span> <span data-ttu-id="b614d-137">Vous trouverez ci-dessous une description de ce fonctionnement.</span><span class="sxs-lookup"><span data-stu-id="b614d-137">The following is a description of how this works.</span></span>

1. <span data-ttu-id="b614d-138">Un chef de projet peut reprojeter l’effort sur les tâches en mettant à jour la valeur par défaut dans le champ **Effort restant** avec une nouvelle estimation de l’effort restant sur la tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-138">A project manager can reproject effort on tasks by updating the default value in the **Remaining Effort** field with a new estimate of the remaining effort on the task.</span></span> <span data-ttu-id="b614d-139">La reprojection entraîne un recalcul de l’estimation de l’effort de la tâche à achèvement (EAA), du pourcentage de progression et de l’écart de l’effort projeté sur une tâche.</span><span class="sxs-lookup"><span data-stu-id="b614d-139">Reprojecting causes a recalculation of the task's effort estimate at complete (EAC), progress percentage, and the projected effort variance on a task.</span></span> <span data-ttu-id="b614d-140">L’Estimation à achèvement (EAC), l’estimation avant achèvement et le pourcentage de progression sur les tâches récapitulatives sont également recalculés, et produisent une nouvelle projection de l’écart d’efforts.</span><span class="sxs-lookup"><span data-stu-id="b614d-140">The EAC, estimate to complete (ETC), and progress percentage on the summary tasks are also recalculated and produce a new projection of effort variance.</span></span>
