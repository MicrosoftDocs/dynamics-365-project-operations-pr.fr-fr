<span data-ttu-id="11c19-108">Les mises à jour de statut suivantes se produisent :</span><span class="sxs-lookup"><span data-stu-id="11c19-108">The following status updates occur:</span></span>

- <span data-ttu-id="11c19-113">Dans l’onglet **Équipe** du projet, la valeur **Statut de demande** du membre de l’équipe générique passe à **Révision nécessaire**.</span><span class="sxs-lookup"><span data-stu-id="11c19-113">On the **Team** tab of the project, the generic team member's **Request Status** value is changed to **Needs Review**.</span></span>

<span data-ttu-id="11c19-115">Le chef du projet peut accepter ou rejeter la proposition.</span><span class="sxs-lookup"><span data-stu-id="11c19-115">The project manager can either accept or reject the proposal.</span></span>

- <span data-ttu-id="11c19-124">Réserver moins de ressources que nécessaire.</span><span class="sxs-lookup"><span data-stu-id="11c19-124">Book fewer resources than are required.</span></span> <span data-ttu-id="11c19-125">Dans ce scénario, les heures réservées sont inférieures aux heures requises.</span><span class="sxs-lookup"><span data-stu-id="11c19-125">In this scenario, the booked hours are fewer than the required hours.</span></span> <span data-ttu-id="11c19-126">Le système vous guide pour proposer des ressources à la place de réservations, de sorte que le demandeur puisse vérifier et suivre la demande restante.</span><span class="sxs-lookup"><span data-stu-id="11c19-126">The system guides you to propose resources instead of bookings, so that the requestor can verify and keep track of remaining demand.</span></span>

## <a name="billable-utilization"></a><span data-ttu-id="11c19-127">Utilisation facturable</span><span class="sxs-lookup"><span data-stu-id="11c19-127">Billable utilization</span></span>

<span data-ttu-id="11c19-128">Les ressources peuvent avoir une utilisation facturable cible.</span><span class="sxs-lookup"><span data-stu-id="11c19-128">Resources can have a target billable utilization.</span></span> <span data-ttu-id="11c19-129">Cette utilisation cible est définie comme un attribut dans le rôle par défaut d’une ressource ou sur l’enregistrement de la ressource réservable individuelle.</span><span class="sxs-lookup"><span data-stu-id="11c19-129">This target utilization is either defined as an attribute on a resource's default role or set on the record of the individual bookable resource.</span></span> <span data-ttu-id="11c19-130">Les calculs de l’utilisation sont basés sur les heures réelles déclarées par des ressources à l’aide d’entrées de temps approuvées.</span><span class="sxs-lookup"><span data-stu-id="11c19-130">Utilization calculations are based on the actual hours that resources have reported by using approved time entries.</span></span>

<span data-ttu-id="11c19-131">Les formules suivantes sont utilisées pour calculer l’utilisation :</span><span class="sxs-lookup"><span data-stu-id="11c19-131">The following formulas are used to calculate utilization:</span></span>
