# <a name="configure-chargeable-components-of-a-project-contract-line"></a><span data-ttu-id="20710-103">Configurer les composants facturables d’une ligne de contrat de projet</span><span class="sxs-lookup"><span data-stu-id="20710-103">Configure chargeable components of a project contract line</span></span>

<span data-ttu-id="20710-104">_**S’applique à :** Project Operations pour les scénarios selon les ressources/produits non stockés_</span><span class="sxs-lookup"><span data-stu-id="20710-104">_**Applies To:** Project Operations for resource/non-stocked based scenarios_</span></span>

<span data-ttu-id="20710-105">Une ligne de contrat basée sur un projet a le concept des composants inclus, facturables et non facturables.</span><span class="sxs-lookup"><span data-stu-id="20710-105">A project-based contract line has the concept of included, chargeable, and non-chargeable components.</span></span>

<span data-ttu-id="20710-106">Les composants inclus sont soumis à la méthode de facturation, aux fractionnements client, aux limites à ne pas dépasser et aux paramètres de fréquence de facturation définis sur la ligne de contrat basée sur le projet.</span><span class="sxs-lookup"><span data-stu-id="20710-106">Included components are subject to the billing method, customer splits, not-to-exceed limits, and invoice frequency settings defined on the project-based contract line.</span></span>

<span data-ttu-id="20710-107">Un sous-ensemble des composants inclus peut être marqué comme payant en mettant à jour le champ **Type de facturation**.</span><span class="sxs-lookup"><span data-stu-id="20710-107">A subset of the included components can be marked as chargeable by updating the **Billing Type** field.</span></span>

<span data-ttu-id="20710-108">Les composants payants peuvent être définis sur les rôles et les catégories de transaction.</span><span class="sxs-lookup"><span data-stu-id="20710-108">Chargeable components can be defined on roles, and transaction categories.</span></span>

<span data-ttu-id="20710-109">Pour une ligne de contrat de projet, la tarification définie sur un rôle s’applique uniquement à la classe de transaction **Temps**.</span><span class="sxs-lookup"><span data-stu-id="20710-109">For a project contract line, the chargeability defined on a role only applies to the **Time** transaction class.</span></span> <span data-ttu-id="20710-110">Si **Inclure le temps** est défini sur **Non** dans une ligne de contrat de projet, l’onglet **Rôles facturables** n’est pas disponible.</span><span class="sxs-lookup"><span data-stu-id="20710-110">If **Include Time** is set to **No** on a project contract line, the **Chargeable roles** tab isn't available.</span></span>

<span data-ttu-id="20710-111">La tarification est définie sur les catégories de transaction pour une ligne de contrat de projet et s’applique uniquement à la classe de transaction **Dépense**.</span><span class="sxs-lookup"><span data-stu-id="20710-111">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="20710-112">Si **Inclure les dépenses** est défini sur **Non** dans une ligne de contrat de projet, l’onglet **Catégories facturables** n’est pas disponible.</span><span class="sxs-lookup"><span data-stu-id="20710-112">If **Include Expenses** is set to **No** on a project contract line, the **Chargeable Categories** tab isn't available.</span></span>

### <a name="update-a-role-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="20710-113">Mettre à jour un rôle pour qu’il soit facturable ou non facturable</span><span class="sxs-lookup"><span data-stu-id="20710-113">Update a role to be chargeable or non-chargeable</span></span>

<span data-ttu-id="20710-114">Un rôle peut être facturable ou non facturable sur une ligne de contrat basée sur un projet spécifique.</span><span class="sxs-lookup"><span data-stu-id="20710-114">A role can be chargeable or non-chargeable on specific project-based contract line.</span></span>

<span data-ttu-id="20710-115">Sous l’onglet **Rôles facturables** de la ligne de contrat basée sur un projet, dans la sous-grille **Catégories facturables**, dans le champ **Type de facturation**, mettez à jour le type de facturation d’un rôle.</span><span class="sxs-lookup"><span data-stu-id="20710-115">On the **Chargeable roles** tab of a projec-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a role.</span></span>

### <a name="update-a-transaction-category-to-be-chargeable-or-non-chargeable"></a><span data-ttu-id="20710-116">Mettre à jour une catégorie de transaction comme facturable ou non facturable</span><span class="sxs-lookup"><span data-stu-id="20710-116">Update a transaction category to be chargeable or non-chargeable</span></span>

<span data-ttu-id="20710-117">Une catégorie de transaction peut être facturable ou non facturable sur une ligne de contrat basée sur un projet spécifique.</span><span class="sxs-lookup"><span data-stu-id="20710-117">A transaction category can be chargeable or non-chargeable on a specific project-based contract line.</span></span>

<span data-ttu-id="20710-118">Sous l’onglet **Catégories facturables** de la ligne de contrat basée sur un projet, dans la sous-grille **Catégories facturables**, dans le champ **Type de facturation**, mettez à jour le type de facturation d’une transaction.</span><span class="sxs-lookup"><span data-stu-id="20710-118">On the **Chargeable Categories** tab of a project-based contract line, on the **Chargeable Categories** subgrid, in the **Billing Type** field, update the billing type for a transaction.</span></span>

### <a name="resolve-chargeability"></a><span data-ttu-id="20710-119">Résoudre la chargeabilité</span><span class="sxs-lookup"><span data-stu-id="20710-119">Resolve chargeability</span></span>

<span data-ttu-id="20710-120">Une estimation ou un réel créé pour le temps ne sera considéré comme facturable que si Temps est inclus dans la ligne de contrat, et si le rôle est configuré comme facturable sur la ligne de contrat.</span><span class="sxs-lookup"><span data-stu-id="20710-120">An estimate or actual created for time will only be considered chargeable if time is included on the contract line, and if the role is configured as chargeable on the contract line.</span></span>

<span data-ttu-id="20710-121">Une estimation ou un réel créé pour la dépense ne sera considéré comme facturable que si la dépense est incluse dans la ligne de contrat, et si la catégorie de transaction est configurée comme facturable sur la ligne de contrat.</span><span class="sxs-lookup"><span data-stu-id="20710-121">An estimate or actual created for expense will only be considered chargeable if expense is included on the contract line, and if the transaction category is configured as chargeable on the contract line.</span></span>
